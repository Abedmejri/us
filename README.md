## Getting Started

### 1. Prerequisites
- Node.js installed.
- MongoDB running locally or a MongoDB Atlas URI.

### 2. Setup Server
1. Go to `server` folder.
2. Create `.env` (already provided with defaults).
3. Run `npm install`.
4. Run `npm start` (or `npm run dev` if you have nodemon).

### 3. Setup Client
1. Go to `client` folder.
2. Run `npm install`.
3. Run `npm run dev`.

### 4. How to Connect
1. Sign up on Page A.
2. Copy your Invite Code.
3. Sign up (or log in) on Page B (or a different browser/incognito).
4. Enter the Invite Code from Page A.
